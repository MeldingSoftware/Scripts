## Mac Scripts

The Mac section contains shell scripts for macOS maintenance, troubleshooting, app tools, system information, terminal utilities, and more.

### Install Mac Scripts

Copy and paste this command into Terminal to install the Melding Software Mac scripts.

```bash
curl -fsSL https://raw.githubusercontent.com/MeldingSoftware/Scripts/main/Mac/install.sh | bash
```

After reopening Terminal, type `MeldingSoftware` anytime to load the list.

## Important Notes

These scripts are provided as-is. Always review scripts before running them, especially scripts that modify system settings, clean files, repair system components, or install software.

Some antivirus programs may flag PowerShell or shell scripts because they automate system tasks. This does not always mean the script is malicious, but you should still review the script before running it.
